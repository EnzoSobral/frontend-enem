# Portal de inscrição - ENEM

## Sobre o Projeto

Este projeto foi desenvolvido utilizando HTML, CSS e JavaScript com o objetivo de simular um portal de inscrição do ENEM. O sistema possui uma tela de login e uma tela de cadastro de candidatos, permitindo a navegação entre as páginas por meio do JavaScript.

A interface foi criada seguindo um padrão visual semelhante aos portais educacionais, com cabeçalho e rodapé fixos, formulário centralizado e layout responsivo para diferentes dispositivos.

## Funcionalidades

- Tela de Login.
- Tela de Cadastro de Candidatos.
- Navegação entre páginas utilizando JavaScript.
- Validação de e-mail contendo o caractere "@";
- Validação de senha com no mínimo 8 caracteres e uma letra maiúscula.
- Confirmação de senha.
- Exibição de mensagens de erro abaixo dos campos.
- Mensagem de sucesso ao concluir o cadastro.
- Redirecionamento automático para a tela inicial após salvar a inscrição.
- Layout responsivo para computadores e dispositivos móveis.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## Objetivo

Aplicar os conceitos de desenvolvimento Front-End aprendidos em sala de aula, utilizando estruturação de páginas com HTML, estilização com CSS, responsividade e validação de formulários através da manipulação do DOM com JavaScript.
